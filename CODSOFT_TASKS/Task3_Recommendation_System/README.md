# Movie Recommendation System (Content-Based)

## Project Overview

This project is a simple web-based Movie Recommendation System developed using Python Flask.

The system follows a **content-based recommendation approach**, where users select a preferred movie category, and the system recommends movies based on that category.

## Features

- User-friendly web interface
- Category-based movie recommendations
- Content-based recommendation approach
- Flask backend integration
- Dynamic display of recommended movies

## Categories Available

The system provides recommendations for:

- Action
- Comedy
- Drama
- Sci-Fi

## Technologies Used

- Python
- Flask
- HTML
- CSS
- JavaScript (Basic Integration)
