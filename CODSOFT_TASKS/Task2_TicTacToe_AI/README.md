# Tic Tac Toe AI Using Minimax

## Project Overview

This project is a web-based Tic Tac Toe game where the user plays against an Artificial Intelligence opponent.

The AI uses the **Minimax Algorithm** to make the best possible moves and prevent losing. The game is developed using Python Flask for the backend and HTML/CSS/JavaScript for the frontend.

## Features

- User plays as **X**
- AI plays as **O**
- AI decision-making using Minimax algorithm
- Interactive web-based interface
- Automatic winner detection
- Draw detection
- Restart game option

## Technologies Used

- Python
- Flask
- HTML
- CSS
- JavaScript
- Minimax Algorithm
