# CODSOFT AI Internship Tasks

## Overview

This repository contains the projects completed as part of the **CODSOFT Artificial Intelligence Internship**.

The projects demonstrate concepts of Artificial Intelligence, Python programming, algorithms, and web application development using Flask.

## Completed Tasks

### Task 1: Rule Based Chatbot

A simple chatbot developed using Python Flask.

### Features:
- User interaction through web interface
- Keyword-based responses
- Predefined conversation flow
- Simple and user-friendly design

Technologies:
- Python
- Flask
- HTML
- CSS
- JavaScript


---

## Task 2: Tic Tac Toe AI Using Minimax

An AI-based Tic Tac Toe game where the user plays against an AI opponent.

The AI uses the **Minimax Algorithm** to find the best possible move.

### Features:
- Player vs AI gameplay
- AI decision making
- Winner detection
- Draw detection
- Interactive web interface

Technologies:
- Python
- Flask
- HTML
- CSS
- JavaScript
- Minimax Algorithm


---

## Task 3: Movie Recommendation System

A content-based movie recommendation system developed using Python Flask.

The system recommends movies based on selected language and genre.

### Features:
- Telugu movie recommendations
- Hindi movie recommendations
- English movie recommendations
- Action, Comedy, and Drama categories
- Dynamic web-based recommendations

Technologies:
- Python
- Flask
- HTML
- CSS
- JavaScript
